Locations

open class Locations

EXPERIMENTAL Ktor feature that allows to handle and construct routes in a typed way.

You have to create data classes/objects representing parametrized routes and annotate them with Location. Then you can register sub-routes and handlers for those locations and create links to them using Locations.href.

readUTF8LineTo

open suspend fun <A : Appendable> readUTF8LineTo(
    out: A,
    limit: Int
): Boolean

Overrides ByteReadChannel.readUTF8LineTo

Reads a line of UTF-8 characters to the specified out buffer up to limit characters. Supports both CR-LF and LF line endings. Throws an exception if the specified limit has been exceeded.

Return true if line has been read (possibly empty) or false if channel has been closed and no characters were read.

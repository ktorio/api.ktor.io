Serializes a value to the specified contentType to a OutgoingContent. This function could ignore value if it is not suitable for conversion and return null so in this case other registered converters could be tried or this function could be invoked with other content types it the converted has been registered multiple times with different content types.

a converted OutgoingContent value, or null if value isn't suitable for this converter
